SOLUTION: Upload Missing .htaccess File

cPanel needs a .htaccess file to exist before it can modify it.

Step 1: Download .htaccess

Download .htaccess file

Step 2: Upload to Correct Location

  1. Open cPanel File Manager
  2. Navigate to /home/simonsta/public_html/
  3. Upload the .htaccess file
  4. Ensure filename starts with a dot: .htaccess

Step 3: Set Permissions

  1. Right-click .htaccess in File Manager
  2. Choose "Change Permissions"
  3. Set to 644
  4. Click "Change Permissions"
